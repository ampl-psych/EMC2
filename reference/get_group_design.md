# Get Group Design

Extracts group design from an emc object

## Usage

``` r
# S3 method for class 'emc.prior'
get_group_design(x)

# S3 method for class 'emc'
get_group_design(x)

get_group_design(x)
```

## Arguments

- x:

  an `emc` or `emc.prior` object

## Value

A design with class emc.group_design
