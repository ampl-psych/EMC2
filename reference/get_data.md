# Get Data

Extracts data from an emc object

## Usage

``` r
# S3 method for class 'emc'
get_data(emc)

get_data(emc)
```

## Arguments

- emc:

  an emc object

## Value

A dataframe of the original data

## Details

emc adds columns and rows to a dataframe in order to facilitate
efficient likelihood calculations. This function will return the data as
provided originally.
