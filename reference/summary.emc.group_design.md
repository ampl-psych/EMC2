# Summary method for emc.group_design objects

Prints a summary of the group_design object For continuous covariates
just prints one row, instead of all covariates.

## Usage

``` r
# S3 method for class 'emc.group_design'
summary(object, ...)
```

## Arguments

- object:

  An object of class `emc.group_design` containing the design to
  summarize

- ...:

  Additional arguments (not used)

## Value

Invisibly returns the design matrices
